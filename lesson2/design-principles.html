Fitt's Law

Item Description
Company 2F COSRX, Inc
Website cosrx.com

According to a Fitts' law the bigger an object and the closer it is to us, the easier it is to move to. I found that invitation to chat with an agent at the bottom of the page is a great call to action.

Visual Hierarchy

Item Description
Company Tesla, Inc
Website tesla.com

You can visually rank the objects on that picture. The first thing that draws attention is the image of the car, the next thing is the name of the tesla model. The third object is a button to "custom order" and another one is to "learn more". We also can see the differences in the colors of these buttons.

Hick's Law

Item Description
Company Kaggle, Inc
Website kaggle.com

Kaggle offers a huge repository of community published data and code. You can find the dataset of your interest by using their filter which very easy to find. Because of that option, it will much easier to navigate and find whatever you are interested in.
